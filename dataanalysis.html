Database Design Summary

Nora’s Bagel Bin Database Blueprints:

Designed a comprehensive database for Nora's Bagel Bin, detailing entities such as Bagel Orders, Bagel Order Line Items, Bagels, and Customers. Implemented the Third Normal Form (3NF) to ensure data integrity and efficient database design. The physical database model includes detailed table structures and relationships for effective data management.

Jauntry’s Coffee Co. ERD:

Developed an ERD for Jauntry’s Coffee Co., defining key entities including Coffee Shops, Employees, Suppliers, and Coffee Products. The schema includes tables and relationships to support coffee shop operations and inventory management, along with SQL queries for data retrieval and manipulation.
